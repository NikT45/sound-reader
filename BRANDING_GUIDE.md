# Resonance Brand Guide

## Brand Vibe
- Keywords: warm, inviting, calm, story-like, audio magic, trustworthy
- Avoid: neon cyberpunk, harsh black-heavy palettes, sharp glass effects, overly technical homepage copy

## Tagline Options
- Hear every page.
- Turn reading into listening.
- Step into the story, gently.

## Color System
Use a light base with aurora accents (blue/violet/aqua) plus a small warm highlight.

### Core Colors
- Resonance Blue (Primary): `#3B82F6`
- Resonance Violet (Secondary): `#7C6EF6`
- Resonance Aqua (Accent): `#45D3E6`
- Warm Glow (Highlight): `#FFD89A`

### Neutrals
- Ink (Headings): `#1F2937`
- Slate (Body): `#475569`
- Muted (Secondary Text): `#64748B`
- Border: `#E6EAF2`
- Surface: `#FFFFFF`
- Canvas: `#F6F8FC`

### Gradient Guidance
- Aurora (Hero background): Violet `#7C6EF6` -> Blue `#3B82F6` -> Aqua `#45D3E6` at 8-16% opacity
- Glow (Featured card): Warm Glow `#FFD89A` at 10-14% opacity
- Rule: keep gradients subtle hints, not full-page overlays

### Softness Rule
- Keep accents slightly desaturated
- Keep backgrounds mostly white
- Prefer gentle contrast over high-tech intensity

## Typography
Choose one system and apply consistently.

### Option A
- Headlines: Plus Jakarta Sans (600-700)
- Body: Inter (400-500)

### Option B
- Headlines: Sora (600-700)
- Body: DM Sans (400-500)

### Type Scale (Landing)
- H1: 56-64 / 1.05
- H2: 36-44 / 1.15
- H3: 24-28 / 1.2
- Body: 16-18 / 1.6
- Small: 13-14 / 1.5

## UI Tokens

### Layout
- Max width: 1120-1200px
- Grid: 12 columns, 24px gutters
- Section padding: 96px desktop / 64px tablet / 48px mobile

### Radius
- Buttons: 12px
- Cards: 16-20px
- Hero container: 28-32px

### Shadows
- Card: `0 12px 30px rgba(31,41,55,0.08)`
- Hover: `0 18px 46px rgba(31,41,55,0.10)`
- Avoid hard shadows and high-contrast borders

### Borders and Surfaces
- Border: 1px `#E6EAF2`
- Card background: white, optional 2-4% blue/violet tint wash

### Background Texture
- Subtle paper/noise texture at 2-4% opacity
- Prefer faint wave arcs (sound-wave motif) over sharp grids

## Component Direction (Landing)

### Header
- Nav: Product, Demo, Pricing, Docs, Blog
- CTA pair: Try it (primary) + See demo (secondary)
- Floating header on a light surface with very subtle blur

### Hero
- Large headline + short subhead
- One strong primary CTA
- Story cards:
  - Upload PDF
  - Choose a Book
  - Press Play
- Add gentle aurora wash behind hero container

### Feature Cards
- 3-6 cards with simple icons:
  - Multi-voice dialogue
  - Emotion control
  - SFX + ambience knobs
  - Shareable audiobooks
- Keep copy human and plainspoken (example: "Make it calmer" / "More cinematic")

### Pricing
- 3 plans; center plan highlighted with warm glow
- Rounded buttons; use Ink/Blue instead of harsh black

### Footer
- Reassure on privacy, safe defaults, and controlled sharing

## Figma Setup Checklist
- Color styles: Blue, Violet, Aqua, Warm, and neutrals
- Text styles: H1, H2, H3, Body, Small
- Effect styles: Card shadow and Hover shadow
- Components: Button (primary/secondary), Card, Nav, Pricing card, Feature card
- Spacing system: 8/16/24/32/48/64/96

## Web Microcopy Tone
- Prefer gentle verbs: listen, bring to life, step into, make it calmer
- Avoid technical jargon on homepage (example: low latency, real-time engine); move that language to technical docs
